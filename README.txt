Tip Calculator by Jinwoo Yom
- This is for a project for a web 
security course in Codepath

How to use:
step 1: enter in the amount in Bill subtotal box
***only positive numerical values are accepted

step 2: choose the tip percentage amount
***10% will be chosen by default

step 3: click the submit button
***The result will appear in a box below
-------------------------------------------
Examples.

inputs : a, aaa, -1, -133, 1da4
*** not valid inputs (positive numerical values only !!)

input : 100
tip   : 10%
output: Tip: $10.00, Total: $110.00

input : 100
tip   : 15%
output: Tip: $15.00, Total: $115.00

input : 100
tip   : 20%
output: Tip: $10.00, Total: $120.00
